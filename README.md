# Tempestuous Savage
A tera-proxy module that automatically uses "Tempestuous Savage Draught" items whenever needed.
 
## Commands
Toolbox(/8) | Command description
--- | ---
**ses** | Module on/off
**ses dungeon** | switch between using items everywhere or only in dungeons
